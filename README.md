# Emission-dashboard
Global Emissioning monitoring dashboard From different Industries and Sectors.
A professional emissions monitoring dashboard with AI-powered insights and interactive data visualizations.
## 1) Features

### Interactive Data Visualization
- **Year Selector**: Compare emissions across 2021-2025
- **Sector Analysis**: Break down emissions by Energy, Transportation, Industry, Agriculture, Buildings, and Waste
- **Regional Distribution**: View emissions by geographic region
- **Trend Analysis**: Visualize historical emission patterns

### AI-Powered Assistant
- Natural language queries about emissions data
- Web search integration for latest climate news and policies
- Quick action buttons for common questions
- Conversational interface for data exploration

### Professional Dashboard
- Real-time metric cards showing key statistics
- Interactive Plotly charts (Bar, Pie, Area)
- Detailed data tables with filtering
- Responsive design for all devices

## 3) Data Coverage

### Sectors Tracked
- Energy Production (Coal, Natural Gas, Oil)
- Industrial Process (Steel, Cement, Chemicals)
- Transportation (Road, Aviation, Shipping)
- Agriculture (Livestock, Crop Production)
- Buildings (Residential, Commercial)
- Waste Management (Landfills, Wastewater)

### Geographic Coverage
- Asia-Pacific: 48.6% of global emissions
- North America: 17.8%
- Europe: 11.0%
- Middle East: 10.2%
- Latin America: 6.3%
- Africa: 3.9%

### Time Period
2021 - 2025 (Annual data with year-over-year comparisons)

## 4) Usage Guide

### Selecting a Year
Use the dropdown selector in the top-right to view data for any year from 2021-2025. All charts and metrics update automatically.

### Using the AI Assistant
1. Scroll to the "AI Emissions Assistant" section
2. Type questions like:
   - "Which sector emits the most?"
   - "Show me the emissions trend"
   - "What are the latest climate policies?"
3. Use quick action buttons in the sidebar for common queries

### Interpreting Data
- **Total Emissions**: Measured in Gigatons (Gt) of CO2 equivalent
- **YoY Change**: Percentage change from previous year
- **Per Capita**: Average emissions per person globally
- **Target Gap**: Difference from 2030 climate goals

## 5) Key Insights

- Global emissions increased 8.2% from 2021 to 2025
- Energy production remains the largest source (35-36% of total)
- Transportation emissions show consistent growth (+11.8% over period)
- Buildings sector shows recent efficiency improvements (-6.3% in 2025)
